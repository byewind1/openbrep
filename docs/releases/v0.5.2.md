# OpenBrep v0.5.2 发布说明

发布日期：2026-03-01

## 本版定位

v0.5.2 不是大功能重构版本，而是一次「版本标注与发布归档规范化」更新，目标是让**代码版本、UI 展示、文档记录**三者保持一致，降低后续版本维护成本。

## 主要变化

### 1) 统一版本号来源（Single Source of Truth）

- 代码版本更新到 `0.5.2`：
  - `openbrep/__init__.py`
  - `pyproject.toml`
- UI 版本展示改为直接读取代码版本变量，避免手工写死造成漂移。

### 2) README 顶部去版本化

- `README.md` 与 `README.zh-CN.md` 标题从 `OpenBrep vX.Y` 改为 `OpenBrep`。
- 版本信息集中放在「版本历史」和发布说明文档中，减少重复维护点。

### 3) 建立发布说明归档

- 新增 `docs/releases/v0.5.2.md`（本文件）。
- 后续建议每个版本都在 `docs/releases/` 下新增一份对应说明，README 仅保留摘要与链接。

## 与功能版本的关系

v0.5.2 重点是规范化和同步，不改变核心业务流程。

此前已落地的重要能力（来自 v0.5.x 演进）包括：

- 2D / 3D 预览接入编辑区
- Tapir Bridge P0：
  - Archicad 实时测试
  - 选中对象 Inspector
  - 参数 Workbench（读取/编辑/回写）

## 仍待推进（面向 v0.6 收口）

- AI 对话结果保存为本地 `.gdl`
- 批量参数重命名并同步脚本引用
