# OpenBrep v0.6.1

发布日期：2026-04-12

## 版本定位

v0.6.1 是基于 `v0.6.0` 的稳定性补丁版本。

它的目标不是开启新的大阶段，而是把 `v0.6.0` 之后已经落地的一批关键改进收口进正式发布口径：补强 CLI 安装后可用性、完善 GDL 静态检查与 repair 闭环、强化 chat / explainer / 图片链路，并同步更新知识规则、打包内容与回归覆盖。

## 核心变化

- CLI 可用性与安装体验提升：`obr` / `obrcli` / help 路径补齐，安装后命令更稳定，并修复 `obr` 在非项目目录无法启动 UI 的问题
- GDL 静态检查与修复链路增强：新增并扩展确定性 linter 规则；生成后补充未初始化变量静态修复；编译失败后自动触发一次 repair
- chat / explainer / 图片链路补强：explainer 对话链路收口；意图澄清路径完善；参考图生成主链接入；终端 chat 的图片路由和引用体验更稳定
- 开源发行内容与知识规则同步：`skills/`、`knowledge/` 进入安装包；补充 GDL 单位系统规则，减少 mm / m 混淆；相关回归测试同步扩展

## 发布判断

- `v0.6.1` 可作为 `v0.6.0` 之后的正式 patch release
- 当前版本重点在于稳定性、可安装性与用户可感知体验收口，而不是引入新的架构阶段
- 包版本、运行时版本、README 与安装文档现已重新对齐，便于后续继续沿 `0.6.x` 迭代
