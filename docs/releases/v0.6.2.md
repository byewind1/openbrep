# OpenBrep v0.6.2

发布日期：2026-04-23

## 版本定位

v0.6.2 是基于 `v0.6.1` 的能力增强与稳定性版本。

本次发布把 knowledge 体系从“内置文档”扩展为“三层知识源”：内置 flat、wiki 概念库、用户自定义 flat，并把 skill creator 的对话式创建流程正式接入 chat 路由，同时补齐关键测试覆盖，确保新链路可回归、可维护。

## 核心变化

- 知识体系升级为三层来源：
  - 内置 flat（`knowledge/GDL_*.md`）
  - wiki 概念库（`knowledge/wiki/*.md`）
  - 用户自定义 flat（`user_knowledge/*.md`）
- chat 链路新增 wiki 知识问答：
  - 支持 GDL 关键词快速命中
  - 支持歧义问题二次分类后检索 wiki 上下文回答
- 新增 skill creator 路由：
  - 支持 `CREATE_SKILL` / `LIST_SKILLS` 意图识别
  - 支持多轮会话与会话结束自动清理
- 新增与补全 GDL wiki 概念页（含 2D/3D 命令、GLOBALS、Object Types 等），提升知识检索覆盖
- 配置与运行时细节修复：
  - `custom provider` 的 `base_url` 回落规则更精确
  - `LLMAdapter` warning filter 迁移到实例初始化，避免测试环境被 reset 后失效
  - 修复 pipeline 中 `Pipeline` 名称引用错误导致的 chat 异常
