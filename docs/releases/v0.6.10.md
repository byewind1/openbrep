# OpenBrep v0.6.10

Release type: macOS installer startup fix.

## Why

The v0.6.9 package smoke run exposed a Streamlit conflict inside the frozen
macOS launcher: packaged builds still saw `global.developmentMode=true`, which
breaks `server.port`. This patch closes that gap.

## What changed

- Packaged launcher:
  - Explicitly sets `STREAMLIT_GLOBAL_DEVELOPMENT_MODE=false`.
  - Keeps the fixed-port and no-browser controls from v0.6.9.
- Smoke tooling:
  - Keeps `scripts/package_smoke.py`.
  - Restores execute bits after zip extraction so the temporary package can run.
- Release docs and version metadata updated to `0.6.10`.
