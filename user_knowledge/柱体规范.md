## 柱体命名规范

- 结构柱：前缀 `STR_COL_`
- 建筑柱：前缀 `ARC_COL_`
- 装饰柱：前缀 `DEC_COL_`

## 常用柱体参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| col_width | Length | 0.4 | 柱宽 |
| col_depth | Length | 0.4 | 柱深 |
| col_height | Length | 3.0 | 柱高 |
| col_offset | Length | 0.0 | 底部偏移 |

## 柱体 GDL 模板

```gdl
! 矩形柱
BLOCK -col_width/2, -col_depth/2, col_offset,
      col_width, col_depth, col_height
```

```gdl
! 圆柱
CYLIND 0, 0, col_width/2, col_width/2, col_height
```
